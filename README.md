# voicegraph-autonomous-agent
VoiceGraph: Autonomous Predictive Outbound Agent — Фреймворк для создания полностью автономных AI-агентов исходящих кампаний (NPS, canvassing, collections). В отличие от классических автообзвонщиков, VoiceGraph использует:  Propensity Modeling, LangGraph Orchestration, Low-Latency Voice Pipeline, Episodic Memory.
